# Vim Practice File

This file is designed to help you practice and improve your Vim skills.
Each section focuses on different Vim techniques and commands.

## Section 1: Basic Navigation

Practice moving around the file using basic navigation commands:
- h, j, k, l - left, down, up, right
- w, b - move forward/backward by word
- e - move to end of word
- 0, $ - move to beginning/end of line
- gg, G - move to beginning/end of file
- :10 - move to line 10

## Section 2: Insert Mode

Practice entering and exiting insert mode:
- i - insert before cursor
- a - insert after cursor
- I - insert at beginning of line
- A - insert at end of line
- o - insert new line below
- O - insert new line above
- Esc - exit insert mode

Try adding text at different positions in these lines:

This line needs text at the beginning.
This line needs text at the end.
This line needs text in the middle.

## Section 3: Deleting Text

Practice deleting text with various commands:
- x - delete character under cursor
- X - delete character before cursor
- dd - delete entire line
- dw - delete word
- d$ - delete to end of line
- d0 - delete to beginning of line
- D - delete to end of line (same as d$)

Delete parts of these lines to fix them:
The quick brown fox jumps over the lazy dog.
The quick brown fox jumps over the lazy dog.
The quick brown fox jumps over the lazy dog.

## Section 4: Yanking (Copying) and Putting (Pasting)

Practice copying and pasting:
- yy - yank (copy) line
- yw - yank word
- y$ - yank to end of line
- p - put (paste) after cursor
- P - put before cursor

Copy text from one line and paste it into another:
Source line: This text should be copied
Target line: Paste the copied text here

## Section 5: Changing Text

Practice changing text:
- cw - change word
- c$ - change to end of line
- cc - change entire line
- r - replace single character
- R - replace multiple characters

Change these lines to say "Hello World":
Good morning everyone
Welcome to Vim practice
Greetings from the editor

## Section 6: Visual Mode

Practice visual selection:
- v - character-wise visual mode
- V - line-wise visual mode
- Ctrl+v - block-wise visual mode
- Use navigation keys to select text
- y - yank selected text
- d - delete selected text
- c - change selected text

Select and modify parts of this text:
This is line one with some text to select
This is line two with more text to select
This is line three with even more text to select

## Section 7: Search and Replace

Practice searching and replacing:
- /pattern - search forward
- ?pattern - search backward
- n - next match
- N - previous match
- :%s/old/new/g - replace all occurrences
- :s/old/new/g - replace in current line

Find and replace words in this paragraph:
The cat sat on the mat. The cat was very happy on the mat.
The dog barked at the cat on the mat. The cat ignored the dog.
The cat continued to sit on the mat all day long.

## Section 8: Working with Multiple Files

Practice working with multiple files:
- :e filename - edit file
- :w - write (save) file
- :q - quit
- :wq or :x - write and quit
- :q! - quit without saving
- :bn - next buffer
- :bp - previous buffer

## Section 9: Advanced Navigation

Practice advanced navigation:
- % - jump to matching bracket
- * - search for word under cursor
- # - search backward for word under cursor
- g_ - go to last non-blank character of line
- fa - find next 'a' character
- ta - find before next 'a' character
- ; - repeat last f, t, F or T command
- , - repeat last f, t, F or T command in opposite direction

Navigate through this code snippet:
function example(a, b) {
  if (a > b) {
    return a;
  } else {
    return b;
  }
}

## Section 10: Macros

Practice recording and playing macros:
- qa - start recording macro 'a'
- q - stop recording
- @a - play macro 'a'
- @@ - repeat last macro

Try recording a macro to format these lines consistently:
this line needs capitalization
this line also needs capitalization
and this one too needs capitalization

## Section 11: Window Management

Practice working with windows:
- :split or :sp - split window horizontally
- :vsplit or :vsp - split window vertically
- Ctrl+w w - switch between windows
- Ctrl+w h/j/k/l - move to window left/below/above/right
- Ctrl+w = - make all windows equal size
- Ctrl+w + - increase window height
- Ctrl+w - - decrease window height

## Section 12: Tabs

Practice working with tabs:
- :tabnew - open new tab
- gt - go to next tab
- gT - go to previous tab
- :tabc - close current tab
- :tabo - close all other tabs

## Section 13: Marks

Practice using marks:
- ma - set mark 'a' at current position
- `a - go to mark 'a'
- 'a - go to line of mark 'a'
- :marks - list all marks
- `` - go to last position
- `. - go to last change

Set marks at different sections and practice jumping between them.

## Section 14: Registers

Practice using registers:
- "ayy - yank line into register 'a'
- "ap - paste from register 'a'
- :reg - show register contents
- " - unnamed register (default for most operations)
- 0 - yank register
- " - small delete register
- + - system clipboard register

## Section 15: Text Objects

Practice using text objects:
- iw - inner word
- aw - a word (including whitespace)
- i( or a( - inside or around parentheses
- i{ or a{ - inside or around braces
- i[ or a[ - inside or around brackets
- i" or a" - inside or around quotes
- it or at - inside or around tags

Try selecting and manipulating these text objects:
"This is a quoted string"
(This is in parentheses)
[This is in brackets]
{This is in braces}
<tag>This is in a tag</tag>

## Section 16: Indentation

Practice indentation commands:
- >> - indent line right
- << - indent line left
- = - auto-indent
- == - auto-indent current line
- =ap - auto-indent paragraph
- =G - auto-indent to end of file
- :set autoindent - turn on autoindent
- :set shiftwidth=4 - set indent width

Format this poorly indented code:
def function():
print("Hello")
if True:
print("World")
else:
print("Goodbye")

## Section 17: Folding

Practice code folding:
- zf - create fold
- zo - open fold
- zc - close fold
- za - toggle fold
- zR - open all folds
- zM - close all folds
- zd - delete fold
- zj - move to next fold
- zk - move to previous fold

## Section 18: Diff Mode

Practice using diff mode:
- vimdiff file1 file2 - open files in diff mode
- ]c - jump to next change
- [c - jump to previous change
- do - diff obtain (get change from other file)
- dp - diff put (put change to other file)
- :diffupdate - update diffs
- :diffoff - turn off diff mode

## Section 19: Command Line Mode

Practice command line commands:
- :!command - run shell command
- :r !command - read output of command into file
- :r filename - read file into current buffer
- :w filename - write to file
- :1,10w filename - write lines 1-10 to file
- :%s/old/new/gc - replace with confirmation

## Section 20: Customization

Practice customizing Vim:
- :e ~/.vimrc - edit vimrc file
- :set option - set option
- :set nooption - unset option
- :syntax on - enable syntax highlighting
- :filetype on - enable filetype detection
- :colorscheme name - change color scheme

Common options to try:
set number
set relativenumber
set tabstop=4
set shiftwidth=4
set expandtab
set autoindent
set smartindent
set hlsearch
set incsearch

## Practice Tips

1. Start with basic navigation and gradually move to advanced topics
2. Practice each section until you feel comfortable
3. Try to use Vim for all your text editing needs
4. Learn one new command or feature each day
5. Use :help in Vim to learn more about any command
6. Create your own practice exercises
7. Time yourself to track improvement
8. Focus on accuracy first, then speed
9. Use Vimtutor for structured learning
10. Have fun and be patient with yourself!
